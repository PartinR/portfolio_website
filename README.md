# Personal Portfolio Website

### Overview
This project is a personal portfolio website designed to outline my skills as a software developer and Computer Science student at Ramapo College. It serves as a central hub for my resume, academic projects, and professional experience.

### Project Goal
To design and implement:
- A lightweight, responsive website built from scratch without heavy frameworks.
- A showcase for technical projects.
- A platform to present my background in finance and software engineering.
